作った関数を利用します。

## コーディング

数値を足し合わせていた場所を、全て作成した関数に置き換えます。
```
suji = add(suji, 90)
```

```
fmt.Println(add(suji,900))
```

実行して同じ出力ができることを確認します。
`clear && go run step4-6.go`{{execute}}

このように出力されます。
```
hello, world.
10
100
1000
```

## 解説

2つの数値の型を受け取る関数なので、2つの数字を`,`区切りで渡します。
```
add(suji, 90)
```

---
1つの出力（数値の型）を返す関数なので、結果を左辺で受け取ります。
```
suji = add(suji, 90)
```
