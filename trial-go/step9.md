複数のデータを管理する配列を使います。<br>
新しくファイルを開きます。

`./step9-10.go`{{open}}

## コーディング

このように記述してください。
```
var a = []int{100, 101, 102}

fmt.Println(a[0])
fmt.Println(a[1])
fmt.Println(a[2])
```

出力を確認します。
`clear && go run step9-10.go`{{execute}}

このように出力されます。
```
100
101
102
```

## 解説

`a`という名前で、`[]int`で数値の型を複数もつ配列変数であることを定義します。<br>
`{}`の中に中身を列挙します。
```
var a = []int{100, 101, 102}
```

---
`[x]`でx番目の中身を参照することができます。<br>
参照は`0`番目から始まります。
```
fmt.Println(a[0])
fmt.Println(a[1])
fmt.Println(a[2])
```
