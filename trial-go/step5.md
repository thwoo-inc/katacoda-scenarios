## コーディング

数値を足し合わせる関数を作成します。
これは`func main()`の手前で独立して定義します。
```
func add(x, y int) int {
  return x + y
}
```

実行してエラーが発生しないことを確認します。
まだ作成した関数は使っていません。

```

```

## 解説

関数の定義を以下のようにしています。
- 2つの入力を数値の型で受け取る（x,yという名前）
- 1つの出力を数値の型で返す

```
func add(x, y int) int {
```

---
2つの入力を足し合わせた結果を、`return`で返しています。
```
return x + y
```
